Output Order

For adjusting the order in the output file (*_Prot) it must be used the option "Output Tags Order" located in the menu Option. There are two options available: with the first the column structure printed in the output file will be: "index" _ "group" _ "aggregation_operator" and if the second is used the structure will be: "group" _ "index" _ "aggregation_operator".
